Lab 7: Firebase Authentication

Lab นี้เป็นการพัฒนาแอปพลิเคชันที่มีระบบรักษาความปลอดภัยและการยืนยันตัวตน (Authentication) โดยใช้ Firebase Authentication ร่วมกับ Ionic React รองรับการเข้าสู่ระบบหลากหลายช่องทาง ได้แก่ Email/Password, Google Login และ Phone Login (OTP) พร้อมการทำ Auth Guard เพื่อป้องกันการเข้าถึงหน้าข้อมูลโดยไม่ได้รับอนุญาต

หลักฐานการดำเนินงาน (5 ขั้นตอนหลัก)

1. หน้า Login หลัก

2. Login ด้วย Email

3. Login ด้วย Google

4. ยืนยันรหัสผ่านทางโทรศัพท์

5. แสดงข้อมูล AuthUser สำเร็จ

ทดสอบแอปพลิเคชันบน Android
